## Известные ограничения

1. **Intra-перевод** — целевая вакансия только в **том же dept + unit**; «С месяца» ≤ месяца перевода; черновик без «Сохранить в план» не в списке.
2. **Seed vs события** — полная синхронизация seed при lifecycle не сделана; пересчёт через `applyEvents` при upsert.
3. **Прогноз** — без наложения событий плана (шаг 6).
4. **Согласования** — заглушка `IN_APPROVAL`.
